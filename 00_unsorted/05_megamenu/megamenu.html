Overview

I've prepared this mega menu example for the purpose of flexibility, reusability and easy integration. If you don't require any customisation on how the mega menu works, you can pretty much rely on the example and just customize the look of it using CSS.

What has been taken care of:

  • Menu Interactions
  • CSS layout and Structure
  • Tested

What you still need to do:

  • Update your content to the given HTML structure
  • Customize CSS styles and layout
  • Set up required parameters by the interaction
X

How it works

This mega menu work as follow:

  • 1st click on menu buttons will open the corresponding mega menu
  • 2nd click on the same menu button closes the mega menu and goes to the specified URL
  • Only one mage menu will be active
  • Click on other menu buttons with mega menu open will switch to other mega menus
  • Click on the close button within the menus will close the active menu
  • Click out of the menus and buttons will close the active menu
  • Only has animation when openning and closing the mega menu, no animation for switching between menus
  • Can have any number of menu buttons and mega menus, as allowed by the container width
  • The menu buttons and menus should pair each other in number

Compatibility

The example is fully tested on:

  • FireFox
  • Safari
  • Chrome
  • IE6*-9

 

* IE6 has minor issues with CSS styles on menu buttons, recommend to use a consistent style for hover and the active states.

X

How to use

You can use this example directly or you can also have your own setup as long as meeting the following points:

HTML and CSS

  • All navigation buttons should share a class name and a parent container
  • All navigation buttons should also share an active state CSS class
  • All mega menu items should share a class name and a parent container
  • All close button in the mega menus should share a class name

jQuery

  • Make sure you supply all the variables in the specified order from the funciton call.
