Data Binding Options

For isolated directives

Text and Two-way binding:

One-way binding:

1
"@" Text binding: returns parsed value using $interpolate from the nearest parent scope, so the expression won't be parsed again against the current scope.
2
"&" Expression binding: expression in attribute value is evaluated against parent scope and retuns as a function to be called in the local(directive) scope, with the option to pass in local data. Depending on the expression, external functions (from parent scope) might or might not get called.
3
"=" Two-way binding: sets a $watch on the current scope member, updates can affect parent members.
