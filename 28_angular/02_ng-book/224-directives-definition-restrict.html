Directives

Definition - Restrict:String

  1. Optional
  2. Defines in which format(s) directives will be declared in DOM
  3. Available values:
    • E - Element name (poor IE support): <my-directive></my-directive>
    • A - Attribute (default): <div my-directive="exp"></div>
    • C - Class: <div class="my-directive: exp;"></div>
    • M - Comment (avoid using, no practical values): <!-- directive: my-directive exp -->
  4. Default value: A (as attributes)
  5. Option values can also be used in combination, e.g. restrict: 'EA' //either element or attribute
  6. Use an element when creating something brand new that will encapsulate a self-contained functionality.
  7. Use an attribute when decorating(enhancing) an existing element with data or behavior.
