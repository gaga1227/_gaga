Built-In Directives

ng-class

ng-attr-(suffix)1

Invalid Attribute Thrown by Browser:

<svg>
  <circle cx="{{cx}}"></circle>
</svg>

The Right Way:

<svg>
  <circle ng-attr-cx="{{cx}}"></circle>
</svg>
1
ng-attr-(suffix) can be used to replace those attributes that are invalid by certain browser. The (suffix) part must be the same as the attribute name.
