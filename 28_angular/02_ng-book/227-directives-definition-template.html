Directives

Definition - Template:String or Function(tElement, tAttrs)

  1. String value needs to be valid HTML string
  2. String value must have a root DOM node/element wrapping texts or multiple childnodes
  3. The function value is required to return a string value representing the template
  4. The t from Function(tElement, tAttrs) stands for template, as opposed to i for instance

Definition - TemplateUrl:String or Function(tElement, tAttrs)

  1. Complex template strings (e.g. multi-line templates) should use templateUrl option instead
  2. The function value is required to return the path to a template file in string
  3. The path to the template file is passed through $sce.getTrustedResourceUrl, a built-in security layer that prevents templates being fetched from untrustworthy places.
  4. Template file is fetched and then cached in the default $templateCache service
